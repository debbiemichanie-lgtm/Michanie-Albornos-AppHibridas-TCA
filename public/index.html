API – Especialistas en Trastornos de la Conducta Alimentaria (TCA)

Esta API permite gestionar especialistas (psicólogas, nutricionistas, psiquiatras, etc.) de Argentina.

Endpoints

  • GET /api/specialists
  • GET /api/specialists?q=ana
  • Filtros

Especialidades

  • GET /api/especialidades (lista; admite ?q=texto)
  • GET /api/especialidades/:id
  • POST /api/especialidades (protegido)
  • PUT /api/especialidades/:id (protegido)
  • DELETE /api/especialidades/:id (protegido)

Auth

POST /api/auth/register
POST /api/auth/login → devuelve JWT
GET  /api/auth/me     (Bearer <token>)
